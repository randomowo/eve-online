# settings

## View Panel

| Panel name | Contains                                |
|------------|-----------------------------------------|
| *          | similar to `General: All` filter        |
| WarpTo     | warp gates, stations and etc.           |
| Target     | PvE, PvP targets, your fleet and etc.   |
| Cargo      | crates and etc.                         |
| Mining     | similar to `Drilling: Drilling` filter. |
| Crew       | your fleet                              |
